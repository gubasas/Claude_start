# Claude_start

A one-command project bootstrap for Claude Code. Run it in any new project folder and it will set up CLAUDE.md, memory, MCP servers, hooks, subagents, and slash commands — automatically.

## Quick Start

**Mac / Linux**
```bash
git clone https://github.com/gubasas/Claude_start.git
cd Claude_start
./install.sh
```

**Windows (PowerShell)**
```powershell
git clone https://github.com/gubasas/Claude_start.git
cd Claude_start
.\install.ps1
```

Then open any new project folder in Claude Code and type `/startnew`.

On your first run you'll be asked to pick your preferred command name — after that it's ready to use in every project.
